1. What is the difference between var, let, and const?
Answer: In JavaScript var, let, and const are all used to declare variables — but they behave differently in scope, hoisting, and reassignment


2. What is the spread operator (...)?

Answer: It is used to expand or spread elements of an array, object, or iterable into individual elements.



3. What is the difference between map(), filter(), and forEach()?
Answer:

map() creates a new array by transforming each item.
filter() creates a new array containing only items that pass a test.
forEach() loops over items but does not return a new array.



4. What is an arrow function?
Answer: Arrow functions provide shorter syntax and handle the this keyword differently than traditional functions.

Syntax:
const sum = (a, b) => a + b;


5. What are template literals?
Answer: Template literals use backticks (`) instead of quotes and allow embedding variables using ${variable}. They are used in this project for dynamic text.
